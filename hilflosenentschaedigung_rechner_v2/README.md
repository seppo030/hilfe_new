# Hilflosenentschädigung‑Funnel (AHV)

Ein mobilfreundlicher Quiz‑Funnel, der Schritt für Schritt die
Anspruchsvoraussetzungen abfragt und am Ende erklärt,
warum (k)ein Anspruch besteht.

## Lokaler Test

```bash
npx serve .
```
